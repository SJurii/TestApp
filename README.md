# Currency Tracker (TestApp) 

Приложение написанное на **C# / WPF** с использованием паттерна **MVVM**. Поддерживает локальное хранение данных в **SQLite** и через **JSON**.

## Как запустить

Если вы хотите просто запустить готовую программу без установки Visual Studio:

1. Скачайте репозиторий.
2. Распакуйте архив.
3. Перейдите в папку `app/Build/`.
4. Запустите файл **`app.exe`**.

---

## Технологии

* **Язык:** C# (.NET 8.0 / 6.0)
* **UI:** WPF (XAML)
* **Архитектура:** MVVM (CommunityToolkit.Mvvm)
* **База данных:** SQLite (Microsoft.Data.Sqlite)

---

## Структура проекта

* `app/Build/` — готовая к работе версия приложения.
* `app/ViewModels/` — логика приложения.
* `app/Models/` — модели данных и работа с БД.
* `app/Views/` — описание интерфейса (XAML).
* `app/Service/` — сервисы для работы с API.
